Array methods

There are different types of arrays methods in javascript

    push
    Adding an element into the array at the end
    pop
    Removing an element from the array at the end
    Unshift
    Adding an element into the array at the initial array
    shift
    Removing an element into the array at the initial array
    splice
    we can add and delete multiple values at a time
